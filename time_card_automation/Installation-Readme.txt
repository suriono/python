
Install Python 3.x : https://www.python.org/downloads/
	Check the box to add to the "Environment Variable"

Download and save the chromedriver in the same folder as the timecardauto.py. https://chromedriver.storage.googleapis.com/2.45/chromedriver_win32.zip

Install Selenium (source: https://pypi.org/project/selenium/)
	pip install selenium 

Install encryption:
	pip install pycryptodome

Click the timecardauto.py to start the time card.
